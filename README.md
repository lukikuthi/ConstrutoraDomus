# Domus Engenharia - Website Institucional

**Autor:** Lucas Kikuthi

## Sobre o Projeto

Website institucional da Domus Engenharia, construtora e incorporadora de alto padrão. Desenvolvido com foco em design sofisticado, performance e experiência do usuário.

## Tecnologias

- **Vite** - Build tool e dev server
- **React 18** - Framework UI
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first CSS
- **Framer Motion** - Animações
- **React Router** - Navegação
- **shadcn/ui** - Componentes UI


## Estrutura do Projeto

```
src/
├── components/     # Componentes reutilizáveis
├── pages/          # Páginas da aplicação
├── data/           # Dados estáticos (projetos, etc)
├── assets/         # Imagens e recursos
├── hooks/          # Custom React hooks
└── lib/            # Utilitários
```
